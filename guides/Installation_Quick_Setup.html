Installation · Quick-Setup

Amp Academy

Every Amp Academy course installs to a phone, tablet, or computer and then runs fully offline — no logins, no accounts, nothing leaves the device. Start at the Campus, install what you need, and you're set. Takes about a minute per device.

Start here — the Campus (your front door)
1

Open the Campus

Go to the link below in a browser. You'll see every course as a card — tap Open → on any one.

https://michaellukasme-dot.github.io/amp-academy/
Install a course to the device
iPhone / iPad
  1. Open the course in Safari
  2. Tap the Share icon
  3. Tap Add to Home Screen
  4. Tap Add
Android
  1. Open the course in Chrome
  2. Tap the ⋮ menu
  3. Tap Install app (or Add to Home screen)
  4. Confirm
Windows / Mac
  1. Open the course in Chrome or Edge
  2. Click the install icon in the address bar
  3. (or ⋮ menu → Install)
  4. Click Install
Confirm it works offline
2

Turn off the internet, open the app

After installing, switch on Airplane mode (or disconnect Wi-Fi) and open the installed course. It loads and runs normally — lessons, the load calculator, quizzes, and the certificate all work with no signal.

About Ask Sader. Everything works offline. The one feature that needs internet is the optional Ask Sader (a live AI helper) — when online, it answers questions and cites the Code (and tells you to confirm against your NFPA 70). When offline, the course still works fully; Ask Sader simply waits for a connection. No student data is stored or collected — Ask Sader only sees the question you type, nothing else.
For a lab or classroom (IT note)
3

Install once per device, or host locally

Install the courses on each student/lab device as above. For fully air-gapped labs, the courses can also be served from a local web server or intranet — see the Admin / IT Deploy & Privacy Guide. No server, account system, or network connection is required for day-to-day use.
